# pi-loom

Unified Pi Loom package.

`packages/pi-loom/` is the sole workspace package in this repository. It contains the former package areas as subdirectories:

- `constitution/`
- `research/`
- `initiatives/`
- `specs/`
- `plans/`
- `ticketing/`
- `critique/`
- `ralph-wiggum/`
- `docs/`
- `storage/`

Load the package locally with:

```bash
cd packages/pi-loom
omp -e .
```

The root `README.md` remains the authoritative architecture and workflow document.
