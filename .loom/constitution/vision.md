## Vision Summary
Build Pi Loom into a repo-visible, local-first, composable coordination and memory system for long-horizon technical work by humans and AI.

## Vision Narrative
Pi Loom turns the ideas in the root CONSTITUTION.md into concrete Loom layers that survive session turnover. It should preserve constitutional policy, research, initiatives, specs, plans, tickets, workers, critique, Ralph orchestration, and documentation as distinct but explicitly linked artifacts under .loom/ so work can be resumed, inspected, critiqued, and improved without transcript archaeology. The project should provide durable primitives and bounded orchestration rather than one rigid workflow, keeping project truth in the repository itself while leaving room for multiple operating modes to emerge from shared building blocks.
